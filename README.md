# Smart Student Management Portal

## Overview
Smart Student Management Portal is a Java-based desktop application developed to simplify and automate the management of student records and academic information. The system provides functionalities for adding, updating, deleting, and searching student details efficiently using JDBC and MySQL database integration.

## Features
- Add new student records
- Update student information
- Delete student data
- Search student records
- Database integration using JDBC
- User-friendly interface

## Tech Stack
- Java
- JDBC
- MySQL
- OOPs Concepts

## Project Objective
The objective of this project is to digitize student data management and reduce manual record handling by providing an efficient and centralized system.

## Modules Used
- Student Record Management
- Database Connectivity
- CRUD Operations
- Exception Handling

- Attendance Management
